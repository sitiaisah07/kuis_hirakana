# 🇯🇵 Hiragana, Katakana & Kanji - Kuis Bahasa Jepang

Aplikasi web sederhana untuk belajar dan menghafal aksara Jepang: **Hiragana**, **Katakana**, dan **Kanji dasar (N5)** — dilengkapi flashcard interaktif, suara pengucapan, dan kuis.

🔗 **Demo:** https://sitiaisah07.github.io/kuis_hirakana/

## ✨ Fitur

- **Flashcard Hiragana & Katakana**
  - Kategori: Dasar, Dakuten, Handakuten, Yōon
  - Klik kartu untuk melihat romaji
  - Tombol 🔊 untuk mendengar pengucapan

- **Kanji N5**
  - 80+ kanji dasar level N5
  - Cara baca on'yomi & kun'yomi
  - Arti dalam Bahasa Indonesia
  - Contoh kata beserta bacaan dan artinya

- **Kuis Interaktif**
  - Pilih kategori soal sesuai keinginan (centang lebih dari satu)
  - Isi romaji (atau arti untuk Kanji), maksimal 2x percobaan per soal
  - Skor akhir dan opsi untuk mengulang kuis

## 🛠️ Teknologi

- HTML, CSS, dan JavaScript murni (tanpa framework)
- Web Speech API untuk pengucapan suara

## 🚀 Cara Menjalankan

1. Clone repository ini:
   ```bash
   git clone https://github.com/sitiaisah07/kuis_hirakana.git
   ```
2. Buka file `index.html` di browser.

Atau langsung akses versi online melalui GitHub Pages di link demo di atas.
